# 누적분포함수

# 정의
- 확률변수 $X$가 어떤 값 $x$ 이하일 확률을 나타내는 함수
- **왼쪽에서부터 쌓아올린 누적 확률**
- $$F(x) = P(X ≤ x)$$

# 성질
- 0에서 1 사이의 값
	- $0 ≤ F(x) ≤ 1$
- **단조 증가 함수**
	- $x$가 커질수록 확률은 줄어들지 않고 커집니다.
	- $a < b$ => $F(a) ≤ F(b)$
- 양극한에서의 값
	- $$\lim_{x\to-\infty}F(x) = 0,  \lim_{x\to+\infty}F(x) = 1$$
- PMF/PDF와의 관계
	- 이산형(PMF)
	- $$F(x) = \sum_{t≤x}{P(X = t)}$$
	- 연속형(PDF)
	- $$F(x) = \int_{-\infty}^{x}{f(t)df}$$
