# 확률밀도함수

# 정의
- 연속확률변수의 확률 분포를 나타내는 함수
- 특정 값의 확률은 0이고, **구간 확률**로만 정의 됨.
- $$ P(a ≤ X ≤ b) = \int_a^b{f(x)dx}$$
- 위 공식에서 $f(x)$가 **PDF**


# 성질
- **비음수성** 
	- $f(x) ≥ 0,$  $∀x$
	- 모든 $x$에 대해, $f(x)$는 0 이상이다.
- **전체 면적 = 1**
	- $$\int_{-\infty}^{\infty}{f(x)dx} = 1$$
- **특정 값의 확률 = 0**
	- $$P(X = a) = \int_a^a{f(x)dx} = 0$$
