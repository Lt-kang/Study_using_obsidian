# 정의
- 태그 이름만 봐서는 콘텐츠의 의미/역할을 알 수 없는 태그
- 의미 전달이 아닌 **레이아웃 / 스타일**을 위해서만 존재하는 태그



# 종류
## div
- Block 단위의 구역을 나누는 컨테이너
```
<div class="container">
  <div class="header">헤더 영역</div>
  <div class="content">본문 내용</div>
  <div class="footer">푸터 영역</div>
</div>
```

## span
- Inline 단위의 구역을 나누는 컨테이너

```
<p>
  이 문장 중 <span style="color:red;">이 부분</span>만 빨간색으로 표시합니다.
</p>
```


# 올바른 사용 전략
|상황|권장 태그|이유|
|---|---|---|
|페이지 전체 구조|시맨틱 태그 (`<header>`, `<main>`, `<footer>`)|의미 중심 설계|
|특정 섹션 내 세부 구획|`<div>`|구조적 구분 및 스타일 지정|
|문장 내 일부 텍스트 스타일링|`<span>`|인라인 스타일 지정|
|의미 없는 레이아웃 보조|`<div>`|단순 시각적 구분용|
