# LiDAR snesor & 측정 원리 
- ToF(Time of Flight)
	- 레이저 발사 -> 반사 -> 수신까지의 시간으로 거리 계산 
	  -> 거리 오차
	  = 시간 오차 x 빛의 속도
- Beam divergence
	- 멀수록 spot size 증가 -> object boundary blur
- Return type
	- Single return
	- Multiple return
- Intensity (반사 강도)
	- 표면 재질
	- 입사각 (grazing angle)
	- 거리 감쇠
	- 센서 gain


# 좌표계 & 기하학
- LiDAR local frame
	- x: forward
	- y: left/right
	- z: up
- World / Vehicle / Camera coordinate
- Extrinsic
	- Rotation (roll, pitch, yaw)
	- Translation
- Projection
	- 3D -> BEV
	- 3D -> Range Image

- ptich 1도만 틀어져도 ground plane 전체가 기울어짐
- BEV height map의 bias 원인 대부분이 extrinsic 문제


# LiDAR 데이터의 통계적 특성 (비정상성 이해)
- distance 증가
	- point density 감소
	- noise 증가
- 각도 의존성
	- grazing angle에서 reflection 불안정
- 공간적 비정상성
	- 동일한 object라도 위치에 따라 통계적 특성이 다름

- threshold 기반 필터링 실패
- 동일 파라미터가 scene마다 다른 결과를 만듦


# Noise & outlier의 종류
## 대표적인 이상치 유형
- random noise
- flying points
- height spike
- multi-path reflection
- reflective surface artifacts(유리, 물)

## 필수 필터링 기법
- SOR(Statistical Outlier Removal)
- ROR(Radius Outlier Removal)
- Ground plane 기반 height filtering
- Temporal consistency filtering


# Ground / Object 분리 개념
- Plane fitting (RANSAC)
- Slope-aware ground model
- Height map 기반 ground filtering
- Progressive TIN densification

- **ground는 평면이 아니라 연속 곡면**
- 오르막/내리막에서 단순 z-threshold는 실패함.

## Auto-labeling과 직결
